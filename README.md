# TripBuzz ✈️

A travel booking website interface built with pure HTML and CSS — no JavaScript, no frameworks.
Inspired by real-world airline platforms like IndiGo, focusing on a clean, modern, and user-friendly layout.

## Preview

![TripBuzz Preview](add-your-screenshot-filename-here.png)

## About the project

TripBuzz is a frontend UI project that replicates the look and feel of a professional airline booking platform.
The goal was to master HTML structure and CSS layout techniques without relying on any JavaScript or external libraries.

## What I focused on

- Pixel-perfect layout inspired by IndiGo and modern airline platforms
- Responsive design using Flexbox and CSS Grid
- Clean navigation bar, hero section, and booking form UI
- Mobile-friendly layout that adapts across screen sizes
- Pure CSS styling — no frameworks like Bootstrap or Tailwind

## What this project taught me

- How to structure complex multi-section layouts in HTML
- CSS Flexbox and Grid for real-world UI patterns
- How professional travel platforms approach UX and visual hierarchy
- Building responsive interfaces without any JavaScript

## How to run

1. Clone the repo or download the files
2. Open the `.html` file in your browser
3. No installs needed — opens instantly
